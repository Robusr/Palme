# Palme🎬
Reinforcement Learning-based Movie Recommendation Platform

## Overview of the questionnaire✍️

### Q1：吸引注意（心理钩子）

**题目：你是否也曾在某个深夜看完一部电影后，坐在屏幕前发呆，觉得刚才那个故事讲的就是你自己？**

**选项：** **[** **是的，深有感触** **] / [** **还没遇到过** **]**

设计意图： 利用“共鸣”瞬间击中用户。大多数电影爱好者都有过这种“被电影读心”的经历，这能瞬间把问卷从“冷冰冰的调查”变成“温暖的灵魂对话”。



### Q2：确认意愿（仪式感引导）

**题目：现在，如果我们能通过** **5** **个直觉选项，在** **30** **种人生剧本中精准抓取你的本命灵魂电影，你愿意开启这场探寻吗？

**选项：** **[** **开启探寻** **] / [** **我再想想** **]**

设计意图： 明确告知用户成本（仅 5 题）和收益（从 30 个项目中精准匹配），并将“填写问卷”动作包装成“开启探寻”的仪式。如果用户点击“开启”，他后续的答题认真度会显著提高。



### Q3：关于电影的“节奏感”，你观影时最喜欢哪种体验？——电影节奏感

**A.** **延迟满足：草蛇灰线，厚积薄发** 哪怕前一个小时的节奏像是在拼一块巨大的拼图，稍显沉闷也没关系。我享受那种铺垫到极致后，在结局瞬间迎来逻辑闭环或情感大爆发的智性愉悦。 *(*指向：慢热剧情*/*史诗*/*硬核科幻，如《沙丘》、《银翼杀手*2049*》、《奥本海默》*)*

**B.** **即时反馈：开局暴击，高频输出** 前十分钟必须用强烈的视觉冲击或核心悬念死死抓住我！我偏爱全程紧凑、绝不拖泥带水的高密度信息轰炸，最好一波未平一波又起，不给大脑留喘息的机会。 *(*指向：动作爽片*/*快节奏悬疑*/*商业巨制，如《疯狂的麦克斯*4*》、《碟中谍》系列*)*

**C.** **张弛有度：呼吸感与正弦波** 我既受不了从头打到尾的神经紧绷，也熬不住过于漫长的文艺留白。我最喜欢“过山车”式的节奏——在紧张刺激的高潮过后，能穿插一段幽默或唯美的文戏让我缓一缓。 *(*指向：经典商业叙事*/*冒险*/*剧情，如《阿凡达》、《楚门的世界》、《头脑特工队》*)*



### Q4：如果把看电影比作一场思维旅行，你最享受导演用哪种方式为你“带路”（剧情推进方式）？

**A.** **流畅沉浸：行云流水的顺叙与共情** 我不喜欢刻意炫技或故弄玄虚，就喜欢导演稳扎稳打地把故事讲透。不管是起承转合的严密推进，还是自然流淌的情绪渲染，只要能让我毫无障碍地代入角色，踏踏实实地沉浸在这段人生里就好。 *(*融合了：流畅推进 *+* 情绪流。受众画像：看重共情与故事完整度的大众影迷。*)*

**B.** **迷雾反转：打破预期的过山车体验** 极度抗拒“一眼看到头”的套路。我偏爱那种藏着掖着、线索一点点抛出的神秘感，或者剧情在一个接一个的意料之外中狂飙。我享受大脑预期被反复颠覆、和编剧“斗智斗勇”的刺激感。 *(*融合了：保持神秘 *+* 频繁反转。受众画像：追求刺激、喜欢悬疑感和心理博弈的观众。*)*

**C.** **智性解构：拼图式的非线性美学** 比起平铺直叙，我更喜欢被打乱的碎片化叙事或多线交织。就像在脑海里拼凑一块巨大的拼图，用严密的逻辑去推演全局。当最后一块线索严丝合缝嵌进去时，那种干净、克制且带有智性美的体验，是最顶级的浪漫。



### Q5：当电影里的危机降临、冲突大爆发时，你最偏爱整部电影呈现出哪种“气质”？——电影套路

**A.** **极致沉浸：智性克制与宏大叙事** 无论是面对沉甸甸的历史宿命（如《奥本海默》），还是用严密逻辑构建的浩瀚宇宙（如《星际穿越》）。我偏爱那种干净、克制、带有智性美的视听语言，在冷峻的写实感或极客式的浪漫中，体会那种被深深震撼、直击灵魂的沉浸感。

**B.** **松弛解构：幽默戏谑与极致爽感** 哪怕天塌下来，主角也能从容地抛个梗。不管是枪林弹雨中“生死看淡，不服就干”的痞帅动作戏（如《速度与激情》），还是用荒诞搞笑化解危机的温情合家欢（如《独行月球》）。我喜欢这种松弛的基调，主打一个看片开心解压、毫无心理负担。

**C.** **疯狂暗黑：黑色幽默与人性深渊** 我不追求传统的“伟光正”，反而迷恋剧情的荒诞感或极端环境下的爆发。不管是在残酷冲突中穿插高级的黑色幽默（如《让子弹飞》、《低俗小说》），还是直面人性幽暗面的心理惊悚（如《小丑》、《消失的爱人》）。这种打破常规、甚至带点神经质的暗黑气质让我觉得极其带感。



### Q6：当一部电影试图触碰你的泪腺时，你最愿意为哪种情绪“买单”？——电影情绪

**A.** **极致的意难平（宣泄型泪水）** 我愿意为那种跨越生死的遗憾、无法挽回的失去痛哭流涕。我不怕被“虐”，反而很享受那种情感在黑暗中被彻底释放、洗涤心灵的宣泄感。 (指向：经典悲剧/深刻爱情/催泪剧情，如《泰坦尼克号》、《寻梦环游记》)

**B.** **绝境逢生的生命力（共振型热泪）** 我不喜欢纯粹的悲剧，但如果是一个小人物在至暗时刻爆发出死磕到底的狠劲，或者在绝望中看到人性的微光，我会因为被深深震撼而“热泪盈眶”。 (指向：高分励志/传记/热血剧情，如《肖申克的救赎》、《当幸福来敲门》)

**C.** **拒绝煽情，把快乐还给我（防御型****/****纯享乐）** 生活已经够累了，我来电影院就是为了精神按摩。我对任何试图让我哭的情节都有防御机制，如果非要流眼泪，我只接受一种可能：笑出眼泪。 (指向：纯喜剧/爆米花爽片/合家欢，如《唐人街探案》、《小鬼当家》)



### Q7：当影院灯光亮起，屏幕上打出“全剧终”的那一刻，你最享受自己走出影院时是哪种状态？观影后主旨带来的感受**

**A.** **灵魂出窍，余味无穷** 脑海里还定格着那个意味深长的绝佳收尾，坐在椅子上久久不想起身。感觉对世界或人生有了新的顿悟，哪怕失眠也要深思到天明。 *(*指向：高分剧情*/*史诗*/*深度科幻，满足*“*产生思考，回味无穷*”)*

**B.** **头脑风暴，细思极恐** 被结尾那个诡异的微笑或惊天反转死死击中！一边感觉脑细胞阵亡了一半，一边疯狂复盘前序细节，爽快地大呼“原来如此，编剧太牛了！” *(*指向：悬疑*/*反转*/*烧脑推理，满足*“*悬疑推理*”)*

**C.** **肾上腺素狂飙，酣畅淋漓** 伴随着主角团嘻嘻哈哈的谢幕或一场极致燃炸的决战，感觉所有的压力都被清空了！心情彻底放松，兴奋到觉得自己现在就能去打十个。 *(*指向：喜剧*/*动作*/*热血，满足*“*感受的兴奋和快乐*”)*
